Goals

  • All colors shall have adequate use
  • Provide general color for UI design
  • All colors will be beautiful in itself and harmonious
  • At the same brightness level, the perceived brightness will be constant

Installation

{% highlight bash %}$ npm install open-color{% endhighlight %}

Variable convention

  • Sass, SCSS: {% highlight scss %}$oc-(color)-(number){% endhighlight %}
  • LESS: {% highlight less %}@oc-(color)-(number){% endhighlight %}
  • Stylus: {% highlight stylus %}oc-(color)-(number){% endhighlight %}
  • CSS: {% highlight css %}--oc-(color)-(number){% endhighlight %}
  • oc Abbreviation for Open color
  • (color) Color name like gray, red, lime ect.
  • (number) 0 to 9. 0 to 9. Brightness spectrum.

How to use

Import the file to your project and use the variables.

Example for Sass, SCSS

{% highlight scss %} @import 'path/open-color'; .body { background-color: $oc-gray-0; color: $oc-gray-7; } a { color: $oc-teal-7; &:hover, &:focus, &:active { color: $oc-indigo-7; } }{% endhighlight %}

Example for LESS

{% highlight scss %} @import 'path/open-color'; .body { background-color: @oc-gray-0; color: @oc-gray-7; } a { color: @oc-teal-7; &:hover, &:focus, &:active { color: @oc-indigo-7; } }{% endhighlight %}

Example for Stylus

{% highlight stylus %} @import 'path/open-color.styl' .body background-color: oc-gray-0 color: oc-gray-7 a color: oc-teal-7; &:hover &:focus &:active color: oc-indigo-7{% endhighlight %}

Example for CSS

{% highlight css %} @import 'path/open-color.css'; .body { background-color: var(--oc-gray-0); color: var(--oc-gray-7); } a { color: var(--oc-teal-7); } a:hover, a:focus, a:active { color: var(--oc-indigo-7); }{% endhighlight %}
